# Portfolio Project

This repository contains a modern portfolio website built with React, TypeScript, Vite, Tailwind CSS, and Framer Motion. It showcases personal branding, an about section, and project highlights including real-world web apps such as e-commerce, business landing pages, and education platforms.

![Demo](Demo.png)

## Overview

The portfolio includes:

- A dynamic hero section with animated text and call-to-action buttons
- An about section with location, education, hobbies, and personal values
- A project showcase featuring multiple live applications and case studies
- Responsive design and smooth section scrolling

## Demo Projects

The app currently highlights these projects:

- **My Shop 3**: A modern e-commerce platform with clean UI, product listings, cart flow, and checkout experience.
- **My Shop 2**: A second iteration of the shop project with enhanced product management and filters.
- **School Jet**: An educational platform for schools with student management, schedules, and teacher dashboard features.
- **Abit Solutions Hub**: A professional business solutions landing page with service display and client-focused layout.
- **Company Website**: A company website with service offerings, team profiles, and contact sections.

## Setup Instructions

### 1. Install dependencies

```bash
npm install
```

### 2. Run the development server

```bash
npm run dev
```

### 3. Open the app

Visit the local URL shown in the terminal, typically:

```text
http://localhost:5173
```

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Available Scripts

- `npm run dev` — Start the Vite development server
- `npm run build` — Create a production build
- `npm run preview` — Preview the production build locally
- `npm run lint` — Run ESLint across the project
- `npm run typecheck` — Run TypeScript type checking
